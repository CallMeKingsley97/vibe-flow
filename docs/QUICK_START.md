# 快速开始

1. 启动 Vibe Flow。应用会扫描支持的 Agent 默认历史目录。
2. 在顶部来源状态中确认 Codex、Claude、Gemini 或 Cursor 已检测到。
3. 从左侧选择会话，先查看“Agent 统计”。
4. 打开“执行流程与事件”，查看阶段梗概、风险节点和完整事件。
5. 点击命令、风险或流程节点可定位到原始事件。
6. 在“设置”中配置本地副本保留周期、查看占用或生成安全诊断包。

默认历史目录：

- Codex：`~/.codex/sessions`、`~/.codex/archived_sessions`
- Claude：`~/.claude/projects`、`~/.claude/transcripts`
- Gemini：`~/.gemini/tmp`、`~/.gemini/chats`
- Cursor：`~/.cursor/projects` 与平台对应的 Cursor User 数据目录

如果来源没有出现，先确认对应 Agent 已产生本地会话，再点击“重新扫描”。
