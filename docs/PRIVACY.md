# 隐私说明

Vibe Flow 默认只处理本机 Agent 已经写入磁盘的历史数据。

## 会读取

- 会话消息和公开推理摘要；
- 工具、Skill、MCP、命令和文件操作记录；
- Agent 历史中的工作目录、时间和错误信息。

## 不会读取或修改

- HTTP/HTTPS 或其他应用网络流量；
- 模型未公开的隐藏思维链；
- 浏览器 Cookie、密码或系统代理配置；
- Agent 原始历史文件的内容与权限。

## 本地存储

统一事件保存在应用自己的 SQLite 数据库中。清理操作只删除该数据库内的副本。诊断包默认只包含版本、设置和数量汇总，不包含消息正文、工具参数、源文件或明文密钥。

## 更新检查

用户主动点击“检查更新”时，应用访问本项目的 GitHub Release 更新端点。更新包必须通过内置公钥验签后才能安装。
