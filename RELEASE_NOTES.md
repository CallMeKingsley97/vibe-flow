# Vibe Flow 0.1.4

聚焦跨会话检索与会话浏览体验，新增全局搜索、收藏与排序能力。

- 跨 Agent 全局搜索：覆盖消息、命令、工具、Skill、MCP、项目等字段，命中高亮并可直接跳转事件；
- 会话收藏：支持收藏过滤与单个会话收藏切换，便于长期关注重要会话；
- 会话排序：新增“最新优先 / 最早优先”切换，对筛选结果复制后按 updatedAt 排序，不修改原数组；
- 会话工具栏重排：搜索框与排序/收藏按钮独占一行，来源标签独占一行，紧凑且不再被换行；
- 后端：新增 0010_session_base_url、0011_session_user_meta 迁移，DTO/contract 对应更新，SQLite 持久化与历史适配器覆盖增强。

升级建议：从 0.1.x 直接安装覆盖即可，本地数据库与设置会保留。

安装前请阅读 [安装说明](docs/INSTALLATION.md) 和 [已知限制](docs/KNOWN_LIMITATIONS.md)。
